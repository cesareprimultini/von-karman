1. Classical Inviscid Model (von Kármán, 1911-12)

Point Vortex Representation

Two infinite parallel rows of point singularities with alternating circulation ±Γ, staggered by half-wavelength.

Velocity Field (Singular)

v(x) = Σi (Γi/2πri) × (x - xi)⊥/|x - xi|

Stability Condition: h/ℓ = 0.281

h = transverse spacing, ℓ = streamwise wavelength

Limitations of Inviscid Model

① Infinite vorticity at centers (singularity)

② No viscous diffusion — pure Euler equations

③ Predicts parallel rows (not observed)

④ No core growth mechanism

Experimental efforts to verify von Kármán's spacing prediction yielded unsatisfactory results — Historical validation

Reality: Vortex rows diverge downstream, cores have finite size and grow via viscous diffusion.

Ref: Royal Society 1936

2. Lamb-Oseen Vortex: Exact Navier-Stokes Solution

Vorticity Distribution (Gaussian)

ωz(r, t) = (Γ / 4πνt) · exp(−r² / 4νt)

Azimuthal Velocity Field

Vθ(r, σ) = (Γ / 2πr) · [1 − exp(−r² / σ²)]

Key Properties:

• No singularity: Vθ → 0 as r → 0

• Far-field: Vθ → Γ/(2πr) as r → ∞

• Circulation Γ conserved (exact N-S solution)

Core Radius Growth (Viscous Diffusion)

σ²(t) = σ₀² + 4νt

Spatial form: σ²(x) = σ₀² + 4νx / U∞

Dimensionless Form

(σ/D)² = (σ₀/D)² + 4(x/D) / Re

Re = U∞D / ν (Reynolds number)

Physical Interpretation: Cores grow as σ ∝ √(x/Re) due to viscous diffusion — validated by experiments at high Re.

Lamb, H. (1932) "Hydrodynamics" • Milne-Thomson (1968)

3. Wake Width Spreading and Vortex Geometry

Linear Wake Spreading Model

h(x) = h₀ · (1 + α · x/D)

h₀ = initial spacing ≈ 0.281ℓ • α = spreading coefficient

Reynolds Number Dependence

• Low Re ~ 100: α ≈ 0.03 – 0.05 (faster spreading)

• High Re ~ 1000: α ≈ 0.01 – 0.02 (slower spreading)

Strouhal Number (Shedding Frequency)

St = f · D / U∞ ≈ 0.2

Constant for 300 < Re < 10⁵

Vortex Street Geometry (Diverging Rows)

Upper Row (+Γ):

xk⁺ = k · ℓ

yk⁺ = +h₀/2 · (1 + α · xk/D)

Lower Row (−Γ), staggered by ℓ/2:

xk⁻ = ℓ/2 + k · ℓ

yk⁻ = −h₀/2 · (1 + α · xk/D)

Circulation per Vortex

Γshed = 2π · St · U∞ · D

Empirical validation: Wake grows linearly when width < integral length scale

4. Dynamic Vortex Advection (Time Integration)

Runge-Kutta 4th Order Integration

xⁿ⁺¹ = xⁿ + (Δt/6)(k₁ + 2k₂ + 2k₃ + k₄)

k₁ = u(tⁿ, xⁿ) | k₂ = u(tⁿ+Δt/2, xⁿ+k₁Δt/2) | k₃ = u(tⁿ+Δt/2, xⁿ+k₂Δt/2) | k₄ = u(tⁿ+Δt, xⁿ+k₃Δt)

Velocity at Vortex Position

u(xi, t) = U∞ + Σj≠i uLO(xi − xj, σj(t))

Self-Induction Regularization

Skip interaction if |xi − xj| < ε · D (typical: ε = 0.01)

RK4 Advantages

4th-order accuracy: O(Δt⁴) local error. Stable without implicit methods.

Simulation Algorithm

for t = 0 to t_max:

① Shed vortex if t ≥ n·Tshed

② Advect all vortices (RK4)

③ Remove vortices outside domain

④ Sample velocity at probes

t += Δt

Computational Complexity

• Per timestep: O(N²) for N vortices

• Total: O(nsteps · N̄²), N̄ ≈ 20-40 typical

Optimization: Fast Multipole Method O(N log N)

Ref: Cottet and Koumoutsakos (2000) "Vortex Methods: Theory and Practice"

5. Comparison: Blevins Model vs Lamb-Oseen Approach

BLEVINS (Empirical/Algebraic)

Velocity Model:

u(y,t) = û · sin(2πy/h) · cos(ωt − φ)

Approach:

• Sinusoidal approximation of wake velocity

• Fixed amplitude decay with distance

• No explicit vortex representation

• Engineering correlation-based

Blevins Pros/Cons

✓ Simple closed-form solution

✓ Fast to evaluate

✗ No viscous physics

✗ Limited spatial validity

LAMB-OSEEN (First Principles)

Velocity Model:

Vθ = (Γ/2πr)[1 − exp(−r²/σ²(t))]

Approach:

• Exact N-S solution for diffusing vortex

• Time-dependent core growth σ(t)

• Superposition of discrete vortices

• Captures vortex interactions

Lamb-Oseen Pros/Cons

✓ Physics-based (exact N-S solution)

✓ Viscous diffusion included

✓ Validated against CFD/experiments

○ Requires time integration O(N²)

Key Difference: Blevins = empirical fit for engineering estimates; Lamb-Oseen = first-principles model for detailed wake physics.

Blevins, R.D. "Flow-Induced Vibration" (2001)

6. Numerical Challenges and Stability Criteria

Critical Numerical Parameters

Δt = 0.01 – 0.05 (constraint: Δt < σ₀/U∞)

ε (cutoff) = 0.01D (constraint: ε ≪ σ₀)

xshed = 0.5 – 1.5D (wake formation length)

yshed = 0.4 – 0.6D (~ D/2 initial separation)

Γshed = 2π·St·U·D (St ≈ 0.2)

Timestep Stability (CFL-like)

Δt < σ₀ / |u|max ≈ 0.1D / U∞

For D=1, U=1: Δt < 0.1, use Δt ≈ 0.02

Validation Checks

① Strouhal Number Recovery

Stmeasured = fpeak·D/U∞ ≈ Stinput (within 5%)

② Circulation Conservation

Σ Γi ≈ 0 (alternating signs cancel)

③ Core Growth Verification

σ(t) = √(σ₀² + 4ν(t − tbirth))

Current Limitations

• 2D flow only (no spanwise variation)

• Fixed separation points (not computed)

• No vortex merging algorithm

Optimization: Fast Multipole Method O(N log N) — Greengard and Rokhlin (1987)
